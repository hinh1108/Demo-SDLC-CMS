# Stakeholder Map + RACI

## Danh sách stakeholder
| Tên / Vai trò | Nhóm | Quan tâm chính | Quyền lực (C/T/T) | Mức quan tâm (C/T/T) | Chiến lược giao tiếp |
|---|---|---|---|---|---|
| | | | | | |

*(C=Cao, T=Trung bình, Th=Thấp)*

## Ma trận RACI
*R=Responsible, A=Accountable (đúng 1/hàng), C=Consulted, I=Informed.*

| Hoạt động / Quyết định | Sponsor | BA | PM | Dev Lead | Người dùng |
|---|---|---|---|---|---|
| Phê duyệt scope | A | R | C | I | C |
| Thu thập yêu cầu | I | R/A | C | C | C |
| Quyết định ưu tiên | A | C | R | C | C |
